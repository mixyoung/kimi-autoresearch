# Internationalization (i18n)

Kimi Autoresearch supports multiple languages.

## Supported Languages

| Code | Language | Status |
|------|----------|--------|
| `en` | English | ✅ Complete |
| `zh` | 简体中文 | ✅ Complete |

## Switch Language

### Command Line

```bash
# Check current language
python scripts/autoresearch_main.py lang

# Switch to English
python scripts/autoresearch_main.py lang en

# Switch to Chinese
python scripts/autoresearch_main.py lang zh
```

### Environment Variable

```bash
# Linux/macOS
export AUTORESEARCH_LANG=zh

# Windows PowerShell
$env:AUTORESEARCH_LANG = "zh"

# Windows CMD
set AUTORESEARCH_LANG=zh
```

### Config File

Create `~/.autoresearch/locale` with the language code:

```bash
echo "zh" > ~/.autoresearch/locale
```

## Priority Order

1. Command line argument
2. Config file (`~/.autoresearch/locale`)
3. Environment variables (`AUTORESEARCH_LANG`, `LANG`, `LANGUAGE`)
4. Default (简体中文)

## Adding a New Language

1. Create translation file:
   ```
   locales/xx/LC_MESSAGES/messages.json
   ```

2. Copy from existing translation and translate

3. Add to `SUPPORTED_LOCALES` in `autoresearch_i18n.py`

4. Test:
   ```bash
   python scripts/autoresearch_i18n.py test
   ```

## Translation File Format

```json
{
  "metadata": {
    "language": "Language Name",
    "code": "xx"
  },
  "messages": {
    "key": "Translation",
    "with_params": "Hello {name}"
  }
}
```

## Common Translation Keys

| Key | English | 中文 |
|-----|---------|------|
| `welcome` | Welcome | 欢迎 |
| `starting` | Starting... | 正在启动... |
| `goal` | Goal | 目标 |
| `success` | Success | 成功 |
| `error` | Error | 错误 |

## Testing Translations

```bash
# Test all languages
python scripts/autoresearch_i18n.py test

# List supported languages
python scripts/autoresearch_i18n.py list
```
